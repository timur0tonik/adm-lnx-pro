# Настройка PXE сервера для автоматической установки

## Цель
Развернуть сервер, обеспечивающий сетевую загрузку и полностью автоматическую установку Ubuntu 24.04 без участия пользователя.


## Реализация
- Использован Vagrant + VirtualBox
- Базовый бокс: Ubuntu 24.04 (собственный `ubuntu24-noble`)
- Используется только Vagrant + shell-провижининг (без Ansible)


## Функционал
- DHCP + TFTP через dnsmasq
- HTTP-сервер через Apache для раздачи:
    - - ядра (vmlinuz)
    - - initrd (initrd)
    - - файлов user-data и meta-data для cloud-init
- GRUB UEFI PXE — поддержка современных систем
- Автоматическая установка с предустановленным пользователем vagrant:vagrant
