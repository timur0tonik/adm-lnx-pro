## Расширенная настройка дисков и сетей

### Окружение
- Стенд поднимается с помощью **Vagrant + Hyper-V**
- Базовый бокс: `myubuntu` (на базе Ubuntu Jammy64)
- Hyper-V в связке с Vagrant **не поддерживает** статическую настройку `private_network` через `ip:`. ВМ получает IP автоматически через DHCP от *Default Switch* 

### Особенности реализации
- Диски реализованы через loop-устройства (ограничение Hyper-V в Vagrant).
- Для проверки сети установлен и запущен `nginx`.
- Созданы два loop-устройства (`/dev/loop0`, `/dev/loop1`) как аналог виртуальных дисков (ограничение Hyper-V).
- Образы размером по 1 ГБ: `/disk-images/disk1.img`, `/disk-images/disk2.img`.
- Отформатированы в `ext4`.
- Смонтированы в `/mnt/disk1` и `/mnt/disk2`.
- Добавлены в `/etc/fstab` по UUID с опцией `nofail`, автоматическое монтирование при загрузке.
- Установлен и запущен `nginx`.
- Тестовая страница: `<h1>Disk & Network HW — nginx works!</h1>`.
