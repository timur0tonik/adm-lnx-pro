# Разворачиваем сетевую лабораторию

## Цель
Научиться менять базовые сетевые настройки в Linux-based системах.


### Серверы
- inetRouter
- centralRouter
- office1Router, office2Router
- centralServer, office1Server, office2Server

Все подсети не пересекаются.

## Реализация
- Использован Vagrant + VirtualBox
- Базовый бокс: Ubuntu 24.04 (собственный `ubuntu24-noble`)
- Все серверы видят друг друга
- Весь трафик в интернет идёт через `inetRouter`
- NAT настроен на `centralRouter` для всех внутренних сетей
